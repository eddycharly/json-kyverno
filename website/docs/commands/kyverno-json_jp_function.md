## kyverno-json jp function

Provides function informations.

### Synopsis

Provides function informations.


```
kyverno-json jp function [function_name]... [flags]
```

### Examples

```
  # List functions
  jp function

  # Get function infos
  jp function truncate

```

### Options

```
  -h, --help   help for function
```
